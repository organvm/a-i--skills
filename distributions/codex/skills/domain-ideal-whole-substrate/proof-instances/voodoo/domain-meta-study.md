## How it was built (process insights)

- **Ontology demanded boundary-statement-first ordering** — typical ontology starts with entities. Voodoo demands "what we mean by this study" come first. Validates that the template's flexibility (sections can be reordered) is a feature, not a flaw.
- **"Heist target" verb broke** — Layer 4 typical semantics fail for religious domains. Required verb-shift to "engage respectfully with" or "cite and extend." Schema *accommodates* this through the verb-shift; it doesn't break.
- **Layer 8 grew to dominant size** — chess Layer 8 is ~10% of substrate weight; voodoo's is ~40%. Both shapes validate schema (it adapts); this is not an overfitting.
- **Anti-extraction guard in Stratum 7 is generative** — what NOT to pull becomes a first-class set of rules, not an appendix. Substrate skill's `domain-attractor-template.md` includes this; voodoo proves it earns its place.

## What it reveals about the meta-system (the substrate skill itself)

- **Schema validation:** all 8 strata fit; load-distribution shifts dramatically. Voodoo's ~40%-Layer-8 weight is the inverse of chess's 10%. Schema is generative across the full spectrum.
- **Operator validation:** all 4 operators fire, but with friction:
  - Selfish-altruistic loop: works only if user is in scholar-respectful mode (not in practitioner mode without initiation)
  - Magnetic membrane: PULL leg has heavy anti-extraction guard; PUSH leg is publication-only (no commercialization)
  - Portfolio Operator: cross-flow with academic-religious-studies domain exists; with commercial domains is dangerous (don't commercially cross-flow voodoo)
  - Reflexive Operator: this very meta-study exists; the recursion holds
- **Verb flexibility:** substrate skill needs to acknowledge that some verbs (e.g., "heist target") are domain-specific and may need shifts for ethical-floor domains. Add to references/8-strata-spec.md.
- **Boundary statement location:** ontology template should *allow* boundary-statement-first when the domain demands it.
