## Usage Notes

1. **Macro-Structure Diagram**: The ASCII diagram should be readable as a standalone map of the script's architecture. All proportions should be roughly to scale.

2. **Model Assessment**: Most scripts are hybrids. Identify the dominant model but note secondary influences.

3. **Dramatic Spine**: The Controlling Idea should be stated as a complete sentence with both VALUE and CAUSE. If you can't articulate it, that's diagnostic.

4. **Movement Breakdown**: Each major section gets its own detailed analysis. Internal structure diagrams should show how beats cluster into sequences.

5. **Turning Points**: The table uses Save the Cat / Hero's Journey terminology as reference points. Not all scripts follow this structure — note absences and assess whether they're problems.

6. **Intensity Mapping**: This should correlate with beat map tension values. Discrepancies between expected and actual intensity curves are diagnostic.

7. **Proportions**: Classical proportions are guidelines, not rules. But significant variance should be noted and assessed.
