# Resume Format Examples

Visual format guidelines and structure examples for different resume types.

## Format Comparison

| Format | Best For | Structure |
|--------|----------|-----------|
| Chronological | Steady career progression | Experience → Education → Skills |
| Functional | Career changers, gaps | Skills → Experience → Education |
| Hybrid | Most professionals | Summary → Skills → Experience → Education |
| Academic CV | Academia, research | Education → Publications → Experience |

## Formatting Guidelines

### Fonts
| Use | Avoid |
|-----|-------|
| Arial, Calibri, Garamond | Comic Sans, decorative fonts |
| 10-12pt body text | < 10pt (hard to read) |
| 14-16pt headers | > 16pt (wastes space) |

### Margins
- Standard: 0.75" – 1" all sides
- Minimum: 0.5" (for more content)
- Maximum: 1.25" (for less content)

### Spacing
- Between sections: 1 blank line
- Between entries: 0.5 blank line
- Line height: 1.0 – 1.15

### File Naming
```
FirstName_LastName_Resume.pdf
FirstName_LastName_Resume_CompanyName.pdf
FirstName_LastName_CV.pdf
```
