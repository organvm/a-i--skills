# Scheduling & Analytics Tools

Tools for managing content distribution, scheduling posts, and tracking performance.

## Scheduling Tools

### All-in-One Platforms

| Tool | Platforms | Free Tier | Best For |
|------|-----------|-----------|----------|
| Buffer | Twitter, LinkedIn, Instagram, Facebook, Pinterest | 3 channels, 10 posts/channel | Solo creators, simple scheduling |
| Hootsuite | All major + YouTube | Limited | Teams, enterprise |
| Later | Instagram, TikTok, Twitter, Facebook, Pinterest | 1 social set, 30 posts/month | Visual-first creators |
| Sprout Social | All major | None (trial only) | Agencies, detailed reporting |
| Publer | All major + Google Business | 3 accounts, 10 scheduled | Budget-conscious teams |

### Platform-Specific Tools

| Tool | Platform | Strength |
|------|----------|----------|
| TweetDeck / X Pro | Twitter/X | Columns, real-time monitoring |
| Creator Studio | Facebook, Instagram | Native scheduling, insights |
| LinkedIn native | LinkedIn | Scheduling, newsletter tools |
| YouTube Studio | YouTube | Upload scheduling, analytics |
| Substack | Email/Newsletter | Built-in subscriber management |

### Choosing a Scheduler

```
Decision tree:

1. How many platforms?
   - 1-2 → Platform-native tools (free)
   - 3+ → Multi-platform scheduler

2. Budget?
   - $0 → Buffer free, native tools
   - $10-30/mo → Buffer, Publer paid tiers
   - $50+/mo → Hootsuite, Sprout Social

3. Team size?
   - Solo → Buffer, Later
   - Small team → Hootsuite, Sprout Social
   - Agency → Sprout Social, Agorapulse
```

## Analytics Tools

### Platform-Native Analytics

| Platform | Built-in Analytics | Key Metrics |
|----------|--------------------|-------------|
| Twitter/X | Analytics dashboard | Impressions, engagement rate, link clicks |
| LinkedIn | Post analytics + page analytics | Impressions, CTR, demographics |
| Instagram | Professional dashboard | Reach, engagement, follower growth |
| YouTube | YouTube Studio | Watch time, CTR, retention |
| Substack | Dashboard | Open rate, click rate, subscriber growth |

### Third-Party Analytics

| Tool | Focus | Free Tier |
|------|-------|-----------|
| Google Analytics | Website traffic, referral sources | Yes (GA4) |
| Plausible | Privacy-friendly web analytics | Trial only |
| SparkToro | Audience research, demographics | 5 searches/month |
| Social Blade | Growth tracking, benchmarks | Basic stats |
| Bitly | Link tracking, click analytics | 10 links/month |

### Key Metrics to Track

```
Awareness Metrics:
- Impressions / Reach
- Follower growth rate
- Share of voice

Engagement Metrics:
- Engagement rate (interactions / impressions)
- Saves and shares (high-value signals)
- Comment quality and sentiment

Conversion Metrics:
- Click-through rate (CTR)
- Email signups from social
- Website visits from social
- Revenue attributed to content

Efficiency Metrics:
- Time per post
- Cost per engagement
- Content repurpose ratio
```

## Recommended Starter Stack

### Zero Budget

```
Scheduling: Buffer free + platform native tools
Analytics: Platform native + Google Analytics
Design: Canva free
Writing: Hemingway Editor (web)
Links: Bitly free
```

### Creator Budget ($25-50/month)

```
Scheduling: Buffer Essentials ($6/channel/mo)
Analytics: Buffer analytics + Google Analytics
Design: Canva Pro ($13/mo)
Writing: Grammarly Premium ($12/mo)
Links: Bitly Starter ($8/mo)
```

### Professional Budget ($100+/month)

```
Scheduling: Hootsuite or Sprout Social
Analytics: Sprout Social reporting + GA4
Design: Canva Pro + Figma
Writing: Grammarly Business
Links: Bitly Business
Research: SparkToro paid
```
