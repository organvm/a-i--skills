# DIWS Proof Instance — Voodoo (outlier validator)

**Source:** No client engagement — pure schema-validation domain
**Created:** 2026-04-25
**Purpose:** Stress-test 8-strata schema on an OUTLIER domain — one that loads layer 8 (external contribution) heavily because of ethical floor concerns.

## Why voodoo?

Voodoo was named in DIWS plan v2.2 as the **schema validator** — if the substrate can describe radically dissimilar domains generatively rather than overfit to commercial / creator-economy patterns, voodoo is the test.

## Filled artifacts

- `domain-ontology.md` — voodoo primitives (lwa pantheon, ritual taxonomy, ceremonial geography)
- `domain-contribution-charter.md` — anti-extraction guard, scholarship contribution, practitioner respect (HEAVY layer 8)
- `domain-meta-study.md` — Reflexive Operator output explaining the layer-8-heavy pattern

## Schema fit observations

- **Layer 1 Ontology** — boundary cases section is *crucial* (boundary between practice and study, between practitioner and outsider, between religion and folk-magic)
- **Layer 4 Gap-map** — entry barrier is HIGH; ethical floor is non-trivial; "heist target" semantics break down (you don't heist religious practice)
- **Layer 8 Contribution** — heaviest load class in any domain; this stratum is half the substrate's weight for voodoo

## Outlier validation result

The schema **fits voodoo** by **shifting load**, not by breaking. Stratum 8 carries roughly 40% of the substrate weight (vs ~10% for chess). The 8 strata still apply; the operators still fire. Schema is generative across radically dissimilar domains. ✓
