# Translation Guide

How to translate ontological names to Latin and Greek.

## Translation Principles

### 1. Concept-Based Translation

Translate the underlying concept, not word-by-word.

```
English: skill-bundle
Concept: "a bundled collection of skills/arts"

Latin: artium-fasciculus
- artium (of arts/skills, genitive plural)
- fasciculus (bundle, small bundle)

Greek: technōn-desmos
- technōn (of arts/skills, genitive plural)
- desmos (bundle, bond)
```

### 2. Preserve Grammatical Relationships

Latin and Greek use case endings to show relationships.

```
English: agent's skill
Latin: agentis ars (genitive + nominative)
Greek: praktoros technē (genitive + nominative)

English: for the agent
Latin: agenti (dative)
Greek: praktori (dative)
```

### 3. Maintain Compound Structure

Keep the hyphen structure, translate each component.

```
English: skill-codex--agent-mastery
Latin:   artium-codex--agentis-magisterium
Greek:   technōn-biblos--praktoros-epistasia
```

## Latin Translation

### Common Vocabulary

| English | Latin (nom.) | Latin (gen.) | Meaning |
|---------|--------------|--------------|---------|
| skill/art | ars | artis | skill, craft, art |
| knowledge | scientia | scientiae | knowledge, science |
| agent | agens | agentis | one who acts |
| forge | fabrica | fabricae | workshop, forge |
| vault | camera | camerae | vault, room |
| codex | codex | codicis | book, codex |
| mastery | magisterium | magisterii | mastery, teaching |
| bundle | fasciculus | fasciculi | small bundle |
| repository | repositorium | repositorii | storehouse |
| power | potestas | potestatis | power, ability |

### Case Usage

| Case | Use | Example |
|------|-----|---------|
| Nominative | Subject, name | `codex` (the codex) |
| Genitive | Possession, "of" | `artium` (of skills) |
| Dative | "for", recipient | `agenti` (for the agent) |
| Accusative | Object | `artem` (the skill) |
| Ablative | "by/with/from" | `arte` (by skill) |

### Compound Formation

```
Pattern: genitive + nominative
"X of Y" → Y (gen.) + X (nom.)

skill-codex → codex of skills → artium codex
- artium (skills, genitive plural)
- codex (codex, nominative)

agent-mastery → mastery of agent → agentis magisterium
- agentis (agent's, genitive singular)
- magisterium (mastery, nominative)
```

### Double-Hyphen Translation

```
skill-codex--agent-mastery
├── skill-codex → artium-codex
├── --          → -- (preserved)
└── agent-mastery → agentis-magisterium

Full: artium-codex--agentis-magisterium
```

## Greek Translation

### Common Vocabulary

| English | Greek (nom.) | Greek (gen.) | Transliteration |
|---------|--------------|--------------|-----------------|
| skill/art | technē | technēs | technique |
| knowledge | epistēmē | epistēmēs | epistemology |
| knowledge | gnōsis | gnōseōs | gnosis |
| agent | praktōr | praktoros | one who acts |
| forge | chalkeion | chalkeiou | smithy, forge |
| vault | thalamos | thalamou | chamber, vault |
| book/codex | biblos | biblou | bible, book |
| mastery | epistasia | epistasias | oversight, mastery |
| bundle | desmos | desmou | bond, bundle |
| power | dynamis | dynameōs | dynamic, power |

### Case Usage

Greek uses similar cases to Latin:

| Case | Use | Example |
|------|-----|---------|
| Nominative | Subject | `technē` (the art) |
| Genitive | "of" | `technēs` (of the art) |
| Dative | "to/for" | `technēi` (for the art) |
| Accusative | Object | `technēn` (the art) |

### Compound Formation

```
Pattern: genitive + nominative
Same as Latin: "X of Y" → Y (gen.) + X (nom.)

skill-codex → biblos of technai → technōn biblos
- technōn (arts, genitive plural)
- biblos (book, nominative)

agent-mastery → epistasia of praktōr → praktoros epistasia
- praktoros (agent's, genitive singular)
- epistasia (mastery, nominative)
```

## Translation Examples

### Example 1: skill-bundle--agent-powers

```
English Analysis:
├── skill-bundle: a bundle containing skills
└── agent-powers: powers belonging to agents

Latin:
├── skill-bundle → artium-fasciculus
│   └── arts (gen.) + bundle (nom.)
├── agent-powers → agentis-potestates
│   └── agent's (gen.) + powers (nom. pl.)
└── Full: artium-fasciculus--agentis-potestates

Greek:
├── skill-bundle → technōn-desmos
│   └── arts (gen.) + bundle (nom.)
├── agent-powers → praktoros-dynameis
│   └── agent's (gen.) + powers (nom. pl.)
└── Full: technōn-desmos--praktoros-dynameis
```

### Example 2: knowledge-forge

```
English: A forge of/for knowledge

Latin: scientia-fabrica
- Could also be: scientiae-fabrica (forge of knowledge)
- Or: scientia-fabrica (knowledge-forge as compound)

Greek: epistēmē-chalkeion
- epistēmē (knowledge) + chalkeion (forge)
```

### Example 3: agent-augment--skill-repository

```
English:
├── agent-augment: augmentation for agents
└── skill-repository: repository of skills

Latin:
├── agent-augment → agentis-augmentum
└── skill-repository → artium-repositorium
Full: agentis-augmentum--artium-repositorium

Greek:
├── agent-augment → praktoros-auxēsis
└── skill-repository → technōn-thēsauros
Full: praktoros-auxēsis--technōn-thēsauros
```

## Pronunciation Guide

### Latin Pronunciation

```
Classical pronunciation:
- c is always hard (like 'k'): codex = "KOH-deks"
- v is like 'w': vault = "WAHL-t"
- ae is like 'eye': scientiae = "ski-en-TEE-eye"

Ecclesiastical (church) pronunciation:
- c before e/i is soft: codex = "KOH-deks" (still hard here)
- ae is like 'ay': scientiae = "shi-en-TEE-ay"
```

### Greek Pronunciation

```
Modern reconstruction:
- ē (eta) is like 'ay': technē = "tekh-NAY"
- ō (omega) is like 'oh': praktōr = "prak-TOHR"
- ch (chi) is like 'kh': chalkeion = "khal-KAY-on"
```

## Validation Checklist
