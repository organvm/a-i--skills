## The Ask

A high-conversion landing page that turns visitors into users/buyers. Short, punchy, focused on transformation.

## Structure

```
1. HERO (above fold)
   - Headline: The transformation
   - Subheadline: How it works in 8 words
   - CTA: Primary button

2. PROBLEM (3 second pitch)
   - What's broken
   - Who it hurts
   - The cost

3. SOLUTION (the product)
   - Core offer
   - What's included
   - No jargon

4. SOCIAL PROOF
   - 3 testimonials
   - 2 case studies
   - Trust signals

5. OFFER
   - Price锚点
   - Bonus bundle
   - Time-limited

6. FAQ (objection handling)
   - 5 common barrier
   - Short answers

7. FINAL CTA
   - One button
   - Urgency anchor
```

## Copy Deck

| Section | What | Words |
|---|---|---|
| Hero headline | "I help [ideal client] achieve [transformation]" | 8 |
| Subheadline | "[method] in [timeframe] without [pain]" | 12 |
| Problem | "Most [ideal client] waste [metric] on [pain]" | 15 |
| Solution | "I use [method] to deliver [transformation]" | 10 |
| Offer | "[Product] at [price]" | 3 |
| CTA | "Start now →" | 2 |

## Conversion Triggers

1. **Specificity** — Numbers over vageness
2. **Urgency** — Time-bound offer
3. **Risk reversal** — Guarantee
4. **Scarcity** — Limited spots
5. **Social proof** — Named people, real results

## Technical Specs

- Single HTML file
- Inline CSS (no dependencies)
- No JS required for core, minimal for analytics
- 100 Lighthouse score target
- Mobile-first, above-fold critical
- Canonical URL + Open Graph
